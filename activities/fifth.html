3. What are special entity characters?

-- Special entity characters are shortcuts to be able to use reserved characters in text. For example, there might be issues when the user types a less than sign because html might interpret it weirdly. To avoid that, the user can use a special entity character to use a less than sign or anything else. < is shown by "& lt ;" (with no spaces)

4. What are phrase tags?

-- Phrase tags are ways to highlight text in various ways, from italics to highlighting to quoteblocks.

5. What is the difference between OL and UL Tags?

-- The difference is the ol tag is ordered, by number letter or numeral. A ul list is unlisted, and only uses bullet points

6. What is a DL?

-- According to w3schools, a dl (description list), is "a list of terms, with a description of each term." The way to mark this is with a dl parent tag, a dt naming tag, and a dd description tag
