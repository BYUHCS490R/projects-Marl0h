Questions!

  • What is JavaScript? How did it get its name?
    -- JavaScript is a coding language for webpage interactivity. It got its name from Sun Microsystems, who wanted to connect their JavaScript language to their Java coding language.
  • What are the two ways you can run JavaScript?
    -- you can embed the JavaScript into the webbrowser/html page (Document Object Model(DOM)). Or, you can use Node.js, which acts like python.
  • What was the most popular front-end js library?
    -- jQuery. it was popular because it handled the different forms of web browser without needing extra code
  • What is the most popular front-end js library today?
    -- React. it is open-sourced from Meta, and is popular for its mobile abilities and other innovations. React can also be used to make webpages into apps for mobile or computer with React native and React Electrum
