CSS

What does CSS stand for?
Cascading Style Sheet. Cascading means that certain details have different priorities. In order, they go like this:
  1. Html Attributes (tags like 'strong' or 'h3'): Limited ability and not used often
  2. Inline CSS (examples below) used within the 'p' tag for example. Not to be used in this class.
  3. Embedded CSS: embedded in the head of the website, also not to be used in this class. Uses the 'style' tag
  4. External CSS: Uses an external file to control the syle of the page. Connected to this page via a 'link' tag in the head of the website. example of use below.

Why should we only use external CSS?

If we use the other forms of CSS, we will override any work that graphic designers put into making the website look good.

What is a selector used for?

A selector is used to decide what tags are changed by the external css. we can select things by their tag name, location and tag, or more ways.
