Questions

  • Why do we need a viewport meta tag?
    -- It is important because it changes the page to make it work on mobile phones and other small screens.
  • When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?
    -- Whichever is the most recent instruction (furthest down the css page).
  • How do you change the CSS for a web page displayed on a phone?
    -- You use an @media tag to change it when the width of the screen reaches mobile sizes.
