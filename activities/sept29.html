Answers to Today's Engagement:

1: List out the different ways to select an item.
You can select the specific tag, like main p, or ID =, or go more general by using the body tag.
2:What is the difference between an ID and a Class?
An Id is a specific identifier that is added to a singular specific tag. A class is similar, but you can use multiple classes in one tag.
3:What is the difference between a SPAN tag and a DIV tag?
A span tag should be used in-line, where a div tag causes linebreaks.
4:What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?
A div tag is a simple divider where the main and other tags are for structure.
5:What are the different ways to select a color?
Using the color: x; command, and choosing a default color, or by inputing an rgb(x,y,z) value. There is also the ability to use hex number and to change the alpha value (tansparency).
