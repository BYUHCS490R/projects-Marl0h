Questions

  • Q1: Describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.
    The parent-child relationship in HTML is when there is a parent tag that determines where block tags go within its are of influence. Any block tag can be a parent tag. The children are the contents within a parent tag that can be moved and placed within the parent block. A parent tag can only directly control the blocks in its influence. It is important for glex and grid because we need to use it to divide the space into different layouts using the parent-child relationships.
  • Q2: What are the two main differences between CSS Grid and CSS Flex?
    CSS Grid is a way to divide a page into a grid format using css and parent-child tags. Grid is 2 dimensional. CSS flex is better built for mobile view, and it allows the child tags to create their own space.
