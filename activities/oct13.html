Questions

  • What is normal flow in HTML?
    -- The normal flow is how things are displayed by default in html, without any changes.
  • What does float do? Why do you need to clear the float?
    -- Float pushes an element to a location you select, then allows for text or other elements to take up space around it. You clear the float so elements you dont want to share space cant, like the footer pushing up
  • What do absolute, fixed, and relative do?
    -- Absolute creates a figure without leaving space for it. Fixed fixes the figure to the view of the page (scrolls with you), and relative creates space for the image, but you can move it relative to the page
  • How can you make nice navigation buttons for mobile devices?
    -- Make the nav options display in block format, and give a background so it is easier to select.
