Internet and Web Vocab & Definitions

Internet

-- The internet is the way that the computers across the world interact with eachother.

Web

-- The web is the content that is shared and communicated between computers.

How are the Internet and the Web different?

-- The internet is the connection between computers, whereas the web is the information that is sent and recieved between users.

ASCII & Unicode

-- ASCII is an encoding format that assigns numerical values to English characters so the computer can display english instead of binary. Unicode is an updated version of ASCII that encodes nearly every language into a digitized format, also allowing easy communication from user to computer.

What is UTF-8

-- UTF-8 stands for Unicode Transformation Format - 8bit. It has become the standard version of Unicode.

URL

-- URL stands for uniform resource locator. It is a reference for specific content that allows users to retrieve information between servers.

Domain Name

-- A domain name is the part of a URL that identifies the service provider of a website.

HTTP

-- HTTP is the foundation for communication on the web, seen at the start of URLs

HTTPS

-- HTTPS is a more secure version of the HTTP format, often displayed with a lock symbol.

IP Address

-- This is the address assigned to your device network as it communicates with other servers.

DNS

-- DNS is a naming system that helps to communicate important web addresses to IP addresses.

W3C Recommendations

-- The W3C is an organization that handles websites and tries to enforce forms of html to create a standard across all websites. A W3C Recommendation is when a site is endorsed by W3c, and is pushed for public viewing when requested.

XML, XHTML, HTML, HTML5

-- These are formats for webpages and the code behind them that are used to display websites.
